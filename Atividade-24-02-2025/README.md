Descrição do Projeto 

   Diante de um trabalho proposto pelo professor Rêmulo do itinerário de Desenvolcimento de Sistemas, eu, 
aluna Giovana Dutra do 3° ano do Ensino Médio da turma de D.S. (Desenvolcimento de Sistemas) finalizei o 
trabalho e realizarei um breve descrição do projeto.
   O trabalho proposto foi o de programar um portfólio pessoal através da programação em HTML, CSS e JavaScript.
   O objetivo do projeto foi criar um portfólio que pudesse ser acessado por qualquer pessoa e que pudesse ser 
   facilmente atualizado. O portfólio deve conter as seguintes informações:

   - Nome do usuário
   - Imagem do usuário
   - Imagens dos programas utilizados no projeto e outros
   - Código do projeto
   - Link para o projeto no GitHub
   - Link para o projeto no repositório do GitHub

   O projeto teve seus desafios, tanto desafios que foram colocados propositamente peloç professor, quanto
desafios que eu mesma enfrentei. O código foi passado pronto para os alunos e nós alunos deveriam realizar
as instruções e corrigir os erros colocados pelo professor. No início tive grande dificuldade em entender o
as linguagens de programação para conçseguir corrigir o código, consequentemente, tive grande ajuda do 
professor e de amigos da turma. Com o tempo, consegui entender melhor as linguagens e realizar o projeto.
   Um desafio que tive, por exemplo, foi o de ajustar a minha foto , que estava muito grande e não estava
aparecendo corretamente no portfólio.
   No dia 9 de março, o projeto foi finalizado com sucesso e aprovado pelo professor Rêmulo.
